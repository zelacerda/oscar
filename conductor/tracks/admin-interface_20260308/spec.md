**Track ID:** admin-interface_20260308
**Type:** Feature

## Summary

Melhorar a interface de admin geral e de bolões com estilos visuais, textos de ajuda em pt-BR informal, e validações de regras de negócio (ex: impedir indicados em categorias incompatíveis).

## Context

O Oscar é uma aplicação web de bolões do Oscar. Já existe uma interface de admin com CRUD básico para todas as entidades (criada no track db-modeling_20260308). Agora é necessário evoluir essa interface para um painel de administração mais completo, com estilo visual definido, textos de ajuda amigáveis e regras de validação de negócio.

## User Story

Como um administrador do sistema, eu quero ter acesso a funções avançadas de administração dos bolões, verificando todos os criadores de bolão, usuários e categorias.

## Acceptance Criteria

- [ ] Interface de administração geral com navegação clara e estilo visual definido
- [ ] Estilo visual da aplicação aplicado consistentemente (cores, tipografia, espaçamentos)
- [ ] Textos de ajuda em pt-BR informal ao longo da interface
- [ ] Requisitos de administração de bolões e usuários (além de CRUD no banco), com regras de negócio como impedir indicados em categorias incompatíveis
- [ ] Validações de regras de inclusão funcionando (ex: indicado de melhor animação não pode aparecer na lista de melhor ator)
