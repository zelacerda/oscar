# Specification: Autenticação com Google OAuth

**Track ID:** google-oauth_20260308
**Type:** Feature
**Created:** 2026-03-08
**Status:** Draft

## Summary

Implementar autenticação via Google OAuth usando NextAuth.js, permitindo que usuários façam login com sua conta Google sem necessidade de criar usuário e senha manualmente.

## Context

O Oscar é uma aplicação de bolões do Oscar onde usuários são convidados por link. Atualmente o cadastro é baseado em usuário e senha. Esta feature substitui esse mecanismo por login social com Google, simplificando a experiência do usuário.

## User Story

Como um participante do bolão, eu quero poder fazer login com minha conta Google para que eu não precise criar e lembrar de mais uma senha.

## Acceptance Criteria

- [ ] Usuário consegue fazer login clicando em "Entrar com Google"
- [ ] Na primeira vez, uma conta é criada automaticamente no sistema (com nome, email e foto do Google)
- [ ] Nas vezes seguintes, o login reconhece o usuário existente
- [ ] Após login, o usuário é redirecionado para a página principal
- [ ] O sistema de autenticação antigo (usuário/senha) é substituído pelo Google OAuth

## Dependencies

- Modelo de usuários existente (será modificado para suportar OAuth)

## Out of Scope

- Outros provedores OAuth (Apple, Facebook, Microsoft)
- Autenticação por usuário/senha (será removida)
- Gerenciamento/edição de perfil do usuário

## Technical Notes

- Usar NextAuth.js (Auth.js) com Google Provider
- Salvar nome, email e foto de perfil vindos do Google no banco de dados
