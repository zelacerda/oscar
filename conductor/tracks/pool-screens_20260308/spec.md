# Specification: Telas de Bolão

**Track ID:** pool-screens_20260308
**Type:** Feature
**Created:** 2026-03-08
**Status:** Draft

## Summary

Implementar as telas de bolão para participantes: tela inicial com lista de membros e acesso a palpites, formulário de edição de palpites por categoria, e tela de ranking em tempo real após o início da cerimônia.

## Context

O Oscar é uma aplicação de bolões onde participantes fazem apostas em categorias antes da cerimônia. As categorias têm pesos diferentes (Ouro=10pts, Prata=5pts, Bronze=3pts, Base=1pt). O sistema bloqueia apostas quando a cerimônia inicia (`lockDate`). Os dados já existem no banco (Pool, PoolMember, Bet, Category, Nominee, Result).

## User Story

Como participante de um bolão, quero acessar uma tela para ver quem está participando comigo, editar meu palpite antes da cerimônia começar e acompanhar o ranking em tempo real quando ela iniciar.

## Acceptance Criteria

- [ ] Tela `/pools/[id]` acessível apenas para membros cadastrados naquele bolão
- [ ] Tela inicial exibe lista de participantes e botão de "Editar Palpite" enquanto `lockDate` não passou
- [ ] Tela `/pools/[id]/bet` com formulário para palpitar em cada categoria, mostrando tier e pontos
- [ ] Palpites podem ser editados livremente até `lockDate`
- [ ] Após `lockDate`, tela inicial exibe link para `/pools/[id]/ranking`
- [ ] Tela de ranking exibe classificação com pontuação atualizada

## Dependencies

- Modelo existente: Pool, PoolMember, Bet, Category, Nominee, Result, User (Prisma)
- APIs existentes: `/api/pools/[id]`, `/api/pool-members`, `/api/bets`, `/api/categories`, `/api/nominees`, `/api/results`
- Autenticação via NextAuth (sessão com userId)

## Out of Scope

- Tela de convite pelo criador do bolão
- Ajustes de interface/design além do necessário para as novas telas
