# Specification: Tela Inicial (Home)

**Track ID:** home-screen_20260308
**Type:** Feature
**Created:** 2026-03-08
**Status:** Draft

## Summary

Tela inicial pos-autenticacao que exibe os boloes do usuario (participante e administrador), com atalho para area de admin e opcao de criar/gerenciar boloes.

## Context

O Oscar e uma aplicacao de boloes do Oscar com autenticacao ja implementada (Google OAuth). Apos o login, o usuario precisa de uma tela central que organize o acesso aos boloes e funcionalidades administrativas.

## User Story

Como um usuario ou administrador, eu quero que apos a autenticacao eu caia numa tela inicial de boas-vindas com informacoes sobre os boloes que eu participo e acesso ao admin.

## Acceptance Criteria

- [ ] Autenticacao redireciona para a tela inicial (Home)
- [ ] Se o usuario for administrador, deve aparecer a barra lateral de admin
- [ ] Se o usuario participa de um ou mais boloes, devem aparecer links para esses boloes
- [ ] Deve aparecer a opcao de criar um novo bolao
- [ ] Se o usuario ja criou um bolao, deve aparecer a opcao de gerar link para convidar membros

## Dependencies

- Autenticacao funcionando (Google OAuth — track `google-oauth_20260308` completa)

## Out of Scope

- Formulario de preenchimento de palpite
- Pagina de convite (apenas prever geracao de um link)
- Telas de acompanhamento do bolao (links podem apontar para paginas inexistentes por enquanto)
