# Specification: Convites e Compartilhamento de Bolão

**Track ID:** pool-invite-share_20260314
**Type:** Feature
**Created:** 2026-03-14
**Status:** Draft

## Summary

Permitir que o criador de um bolão adicione uma descrição breve, compartilhe convites via link ou Web Share API com mensagem personalizada, e oferecer uma tela dedicada de convite que direciona ao login e adiciona o usuário automaticamente ao bolão.

## Context

O Oscar já possui um sistema de bolões com criação, apostas e ranking. Existe um `inviteCode` no modelo Pool e um botão de copiar link, mas não há rota `/invite/[code]` nem fluxo de entrada via convite. A autenticação usa Google OAuth via NextAuth.

## User Story

Como **criador de bolão**, eu quero adicionar uma descrição ao bolão e compartilhar convites de forma prática (link ou share nativo) para que os convidados tenham uma experiência fluida de entrada no bolão, com uma tela de convite dedicada que já os direciona ao login/cadastro e os adiciona automaticamente ao bolão.

## Acceptance Criteria

- [ ] Criador pode adicionar uma descrição breve ao bolão (na criação e edição)
- [ ] Botão "Copiar link" copia apenas a URL de convite para a área de transferência
- [ ] Botão "Compartilhar" usa a Web Share API no mobile com mensagem personalizada (nome, descrição e link)
- [ ] URL de convite leva a uma tela especial de login/cadastro com contexto do bolão
- [ ] Se o usuário já estiver logado ao acessar o link de convite, ele é adicionado automaticamente ao bolão

## Dependencies

- Sistema de bolões existente (track `pool-screens`)
- Autenticação Google OAuth (track `google-oauth`)

## Out of Scope

- Notificações push ou por e-mail sobre convites
- Sistema de convites com aprovação do admin (convite direto, sem moderação)
- Expiração de links de convite
- Personalização visual da mensagem de compartilhamento além do template padrão
